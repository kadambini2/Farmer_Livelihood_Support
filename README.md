# 🌾 Sahayak AI – Climate-Resilient Livelihood Planner

## 📌 Overview

**Sahayak AI** is a **Streamlit-based, multilingual, voice-enabled web application** designed to help farmers in **dry and drought-prone regions of India** explore secondary income opportunities and adopt climate-resilient practices. The app provides **personalized recommendations** based on environmental conditions, farm resources, and farmer preferences.  

The tool is **inclusive**, requiring **no login**, and works for both **literate and illiterate farmers** with text and voice input/output.

---

## 🎯 Objectives

* Provide personalized livelihood planning for farmers in dry regions  
* Suggest **secondary income streams** like poultry, beekeeping, mushroom cultivation, inland fisheries, and agro-processing  
* Promote **climate-resilient agriculture** and sustainable practices  
* Enable **zero-waste utilization** of crops (e.g., coconut, millet, jute)  
* Provide a **simple, multilingual, and voice-enabled interface**  

---

## 🧑‍💻 Tech Stack

* **Frontend & Backend:** Streamlit  
* **Language:** Python  
* **Libraries:** Pandas, NumPy, gTTS (Text-to-Speech), optional: Scikit-learn for recommendations  
* **Voice Input (Future):** Google Speech Recognition / Web Speech API  
* **Deployment:** Streamlit Cloud / Local Server  

---

## ⚙️ Features

* 🌱 **Farmer Details Input** – Region, land size, water availability, budget, optional crop preference  
* 🎙 **Voice + Text Input** – Supports illiterate and literate users  
* 🌐 **Multilingual Support** – Kannada, Hindi, Marathi, Gujarati, Telugu, Rajasthani, Assamese, Odia, Tamil, English  
* ✅ **Instant Confirmation** – Summarizes inputs via text and TTS before recommendations  
* 💡 **Recommendations** – Suggests suitable crops, side ventures, and resource utilization strategies  
* 📊 **Simple Visual Insights** – Icons and summaries for easy understanding  
* 🧭 **Inclusive Design** – Mobile-first, no login, easy navigation  

---
